# templates — 所有 Prompt 與模板檔案

## 用途
存放系統使用的所有 Prompt 定義與文件模板。

## 存放內容

### Prompt 模板
- `prompt_planner.txt`：Planner Agent 角色定義與規則
- `prompt_executor.txt`：Executor Agent 角色定義與規則
- `prompt_reviewer.txt`：Reviewer Agent 角色定義與規則

### 文件模板
- `tasks_template.md`：通用任務清單模板
- `acceptance_template.md`：驗收標準模板
- `assumptions_template.md`：假設與風險登錄模板
- `glossary_template.md`：術語表模板
- `change_log_template.md`：版本變更記錄模板

### 專用任務模板
- `tasks_governance.md`：治理文件產出任務樹
- `tasks_sysdesign.md`：系統設計 Baseline 任務樹
- `tasks_compliance.md`：合規稽核矩陣（IEC 62443）任務樹
