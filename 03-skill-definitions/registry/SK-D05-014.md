## Description

Design frequency regulation control algorithms for grid-connected generation and energy storage assets, including droop control, Automatic Generation Control (AGC) participation, and synthetic inertia provision from inverter-based resources (IBRs). This skill addresses the technical challenges of maintaining grid frequency stability as traditional synchronous generation is replaced by asynchronous renewables and distributed generation. Design covers primary frequency response (droop control, inertial response), secondary frequency control (AGC participation), and support for voltage stability. The output is a detailed specification of frequency regulation algorithms and parameters that guides control implementation in the EMS/DERMS system and inverter firmware. This skill feeds into SK-D05-003 ⏳ (EMS AGC Configuration) for actual AGC tuning and commissioning.

## Inputs

- Electrical Network Architecture and power system model (from SK-D02-001 ⏳: OT Network Topology Design, specialized for generation/storage)
- System inertia analysis and frequency stability assessment (baseline and post-DG scenarios)
- Generation and energy storage asset specifications: power ratings, ramp rates, frequency response capabilities, control interfaces
- Distribution/transmission grid frequency requirements and standards (e.g., NERC frequency response requirements, local grid code requirements)
- Historical frequency data and disturbance scenarios (contingency analysis results)
- System Architecture Design and EMS/DERMS Functional Specification (from SK-D05-001 ⏳ and SK-D05-002 ⏳)
- Inverter and DER control capabilities (available control modes, communication protocols, response latency)
- AGC and economic dispatch requirements from the grid operator

## Outputs

- Frequency Regulation Strategy Document: overall framework for frequency control including primary response targets, AGC participation model, inertial response requirements, and voltage support coordination
- Primary Frequency Response Specification: droop control settings for synchronous and inverter-based generation, including gains (droop slope in Hz/MW), deadbands, ramp rate limits
- Synthetic Inertia Specification: virtual inertia algorithms for IBRs, including inertial response gain, damping ratio, and rate-of-change-of-frequency (RoCoF) detection thresholds
- AGC Algorithm Specification: AGC participation rules, economic dispatch methodology, unit commitment constraints, and interaction with primary frequency control (e.g., integration with damping and droop)
- Control Tuning and Validation Plan: procedure for determining optimal control parameters through simulation, hardware-in-loop testing, and field validation
- Interface Specification: communication requirements and data exchange for frequency control between assets, inverter controllers, and EMS/DERMS
- Performance Validation Criteria: frequency response metrics (settling time, overshoot, stability margin) against grid code requirements

## Tools

- MATLAB/Simulink (power system simulation, control algorithm modeling, frequency response simulation per IEEE STD 3002 procedures)
- PSS/E / ETAP (large-scale system frequency studies and contingency analysis)
- TYPHOON HIL / RTDS (real-time simulation for control validation and hardware-in-loop testing)
- Python / Jupyter (data analysis of historical frequency events, parameter tuning optimization)
- Inverter simulation tools (vendor-specific simulation for IBR control validation)
- D2 / Simulink diagrams (control architecture and signal flow)

## Standards

- IEEE Std 3002 — IEEE Guide for Testing and Evaluation of Dynamic Performance of Microgrid Controls: frequency control testing methodology
- NERC FROR (Frequency Response Obligation Requirements) and EOP standards: frequency response performance requirements for grid operators
- IEC 61000-4-27 / IEEE STD 1159 — Power Quality — frequency variations and transient response standards
- IEC 61727 / IEEE 1547 — Distributed Generation — DG frequency and voltage control requirements
- WECC Dynamic Security Assessment standards and NERC TPL-001 — Transmission Planning standards relevant to frequency stability
- IEC 62443-3-3 — System Security Requirements and Security Levels: security of AGC and control signals
- IEEE Std 2030.7 — Guide for the Interoperability of Energy Technology and Information Technology Operations with the Electrical Power System (EPS)

## IEC 62443 Lifecycle Stages

- R2 (Detailed Design — frequency regulation algorithms and control parameters finalized through simulation)
- R3 (Implementation — control algorithms implemented in EMS/DERMS and inverter firmware, validated through field testing)

## Roles

- SYS (System Architect) — primary: develops overall frequency regulation strategy, AGC algorithm, and control architecture
- Control Engineer — supports with algorithm design, tuning methodology, and validation testing
- Electrical Engineer / Power Systems Specialist — supports with frequency stability analysis and grid code compliance assessment
- Inverter/DER Controls Specialist — designs IBR frequency response algorithms and validates compatibility with inverter hardware
- EMS/DERMS Specialist — integrates AGC and frequency control into system operation
- STC (Security Technical Consultant) — reviews frequency control for security implications (e.g., authentication of control signals, denial-of-service resilience)

## Dependencies

- SK-D05-001 ⏳: SCADA/EMS/DERMS Functional and Software Design — cannot design frequency control algorithms without knowing what EMS functions will support AGC and control dispatch
- Electrical system model and frequency stability analysis — frequency control design depends on detailed understanding of system inertia and frequency response requirements

### Soft Dependencies (Enhances Effectiveness)

- SK-D05-002 ⏳: SCADA/EMS/DERMS Detailed Software and Control Design — frequency control algorithms inform control implementation
- SK-D05-013: Load Management Strategy Design — load shedding and demand response interact with frequency regulation; coordination may improve overall stability
- SK-D05-003 ⏳: EMS AGC Configuration — actual AGC parameter tuning and field validation builds on this design
- SK-D05-014 itself feeds SK-D05-003 ⏳ for AGC tuning and commissioning

## Acceptance Criteria

1. Frequency regulation strategy clearly defines primary response targets (droop gain, response time), AGC participation model, and synthetic inertia requirements
2. Primary frequency response specification includes droop control settings for all generation/storage assets with documented rationale based on frequency stability analysis
3. Synthetic inertia algorithm is specified for IBRs with clear inertial response gain and damping parameters
4. AGC algorithm specification includes unit commitment logic, dispatch constraints, and interaction rules with primary frequency control
5. Simulation results demonstrate frequency response performance meets grid code requirements under defined contingency scenarios (e.g., loss of largest generator)
6. Control parameters are derived from validated power system simulation (MATLAB, PSS/E, or equivalent)
7. Communication interface specification is complete: data tags, exchange frequency, timing, and synchronization requirements for AGC dispatch
8. Hardware-in-loop or real-time simulation validation plan is defined and feasible
9. Design has been reviewed and approved by SYS, control engineer, electrical engineer/power systems specialist, and STC
