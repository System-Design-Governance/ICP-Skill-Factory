## Description

Calculate distance relay settings (Zone 1/2/3 reach, time delays) for transmission and sub-transmission line protection using impedance-based calculations derived from power system models. This skill involves determining relay reach points on the R-X plane, calculating time delays for multi-zone protection, and validating coordination with adjacent protection schemes. Uses impedance calculations from the power system model (SK-D03-010) and supports selective fault isolation on long-distance transmission corridors. Results feed into protection logic diagram development (SK-D04-004) and SCADA alarm configuration.

---

## Inputs

- Power system impedance model (source impedances, line impedances, transformer impedances)
- Transmission line positive/zero-sequence impedances and distribution
- Fault study results (maximum and minimum fault currents, SIR values)
- Distance relay manufacturer specifications and user manuals
- Network expansion plans (future generator/load connection points)
- Coordination requirements with adjacent substations (voltage levels, relay types)
- System grounding configuration (solidly grounded, resistance-grounded, isolated neutral)

---

## Outputs

- Distance relay setting calculation report
- Zone reach settings (Zone 1: typically 80% of line length; Zone 2: 120% + margin; Zone 3: backup reach)
- Time delay settings for each zone (instantaneous for Zone 1, delayed for Zone 2/3)
- R-X plane plots showing relay characteristics and coordination margins
- Sensitivity analysis (SIR margin, load encroachment risk assessment)
- Setting verification checklist and relay parameter export files
- Contingency analysis (impact of adjacent line outage on relay reach)

## Acceptance Criteria

1. Zone 1 reach set to 80–90% of protected line impedance (eliminating infeed bias)
2. Zone 2 reach set to 120–150% of protected line impedance with adequate margin to adjacent zone
3. Zone 3 reach provides backup protection to adjacent line (typically 50–75% of adjacent line impedance)
4. Time delays comply with coordination intervals (Zone 2: 0.3–0.5 sec delay; Zone 3: 1.0–2.0 sec delay)
5. R-X characteristic plot demonstrates no load encroachment under maximum load flow scenario
6. Sensitivity ratio (SIR) margin maintained above minimum threshold (typically >1.5 for thermal generators)
7. Relay setting parameters exported and cross-checked against manufacturer specifications
8. Contingency analysis confirms reach adequacy if adjacent protection zone is unavailable
