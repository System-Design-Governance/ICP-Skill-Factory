## Description

Execute comprehensive protection relay testing on installed relays in a power generation or distribution system, validating their correct configuration, operation, and coordination with downstream devices. This skill covers primary injection testing (energized bus testing), secondary injection testing (test transformer injection), trip timing verification, and communication testing (IEC 61850 GOOSE protocol). Testing validates relay settings derived from SK-D04-001 (Overcurrent Protection) and SK-D04-003 (Relay Selection), ensuring relays respond correctly to fault conditions and maintain proper coordination with adjacent protection zones. Testing uses specialized relay test sets (Omicron, Doble, Schweitzer) and follows IEEE/IEC testing standards. This skill is part of the Factory Acceptance Test (FAT) and Site Acceptance Test (SAT) phases, feeding into commissioning reports (SK-D07-001).

## Inputs

- Relay settings from SK-D04-001 (Overcurrent Protection Settings)
- Relay selection and specification from SK-D04-003 (Relay Selection)
- Approved protection coordination curves (from SK-D04-002 ⏳: Protection Coordination Design)
- Relay test plans with acceptance criteria (from SK-D07-003 ⏳: Test Plan Development)
- Relay documentation (IED specification sheets, communication protocol definitions)
- As-built relay configuration files (ICD, SCD per IEC 61850)
- Factory/field test infrastructure (test sets, current transformers, voltage dividers)

## Outputs

- Relay Test Report (primary and secondary injection results, trip time measurements)
- GOOSE Communication Test Report (IEC 61850 signal transmission and timing validation)
- Relay Coordination Verification Report (selectivity confirmation with adjacent zones)
- Test data sheets (logged test points, actual vs. expected trip times)
- Sign-off documentation and FAT/SAT acceptance sign-off sheets
- Discrepancy reports and corrective action log for any failed tests

## Tools

- Omicron CMC 356 or equivalent relay test set (primary/secondary injection)
- Doble M4100 or equivalent portable relay test equipment
- IEC 61850 protocol analyzer (for GOOSE communication testing)
- Relay parameter extraction tools (CAPE, EasePower)
- Oscilloscope / transient data recorder (for high-speed timing verification)
- Excel / structured markdown (test data recording)

## Acceptance Criteria

1. All relay test points (pickup, drop-out, timing) are within manufacturer specification and project acceptance tolerance (typically ±3% for electromechanical, ±2% for static relays)
2. Trip times are verified within coordination window (typically 200–300 ms below next upstream device)
3. GOOSE communication from relay to RTU/SCADA is transmitted and received within specified latency (typically ≤4 ms per IEC 61850-8-1)
4. Relay coordination with adjacent protection zones confirmed: selectivity maintained for all zone-to-zone fault scenarios
5. All test equipment is within current calibration (certificates provided)
6. Test report is signed by DES and approved by QA/ELE
7. Zero critical discrepancies; any open issues documented with corrective action plan

| Senior (5+ yr) | 1–3 person-days per relay | Can optimize test sequences, diagnose communication issues independently |

Notes: Effort scales with number of relays, complexity of communication protocols (IEC 61850 GOOSE adds ~1–2 days overhead per relay). Multi-function relays (e.g., combined overcurrent + distance) may require 1.5–2× time. Brownfield sites with undocumented relay settings may require additional settings audit.
