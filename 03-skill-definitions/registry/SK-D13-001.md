## Description

The Automated Calculation Script Development skill develops and deploys reusable calculation scripts (Python, MATLAB, Excel VBA) that automate repetitive engineering calculations essential to power system and OT/ICS design. This capability addresses cable sizing, voltage drop analysis, protection coordination, thermal analysis, and load flow parameter extraction—calculations that are critical for system safety and often error-prone when performed manually. Scripts incorporate input validation, calculation verification, and comprehensive output documentation to ensure calculation accuracy and traceability. By automating these engineering workflows, this skill reduces design cycle time, minimizes human error, and enables rapid sensitivity analysis and design iteration while maintaining confidence in electrical safety calculations.

## Inputs

- Engineering calculation methodologies and technical standards (IEC 60287, IEEE 45, IEC 60050)
- Cable and equipment specifications (conductors, insulation, ratings)
- System parameters (voltages, currents, fault levels, duty cycles)
- Protection device data (coordination curves, settings, time-current characteristics)
- Load flow analysis data and thermal boundary conditions
- Design constraints and acceptance criteria

## Outputs

- Calculation scripts (Python, MATLAB, or VBA implementations)
- Input specification and validation procedures
- Calculation methodology documentation (technical approach and formula justification)
- Verification and testing procedures (unit tests, regression tests)
- Output templates and example calculations
- User guide and troubleshooting documentation

## Acceptance Criteria

1. Scripts implement all required calculation types (cable sizing, voltage drop, protection coordination, thermal analysis, load flow) with full traceability to applicable standards
2. Input validation rejects out-of-range or malformed inputs with clear error messages
3. Script results match manually calculated results within engineering tolerance (typically 1-2% for electrical calculations)
4. Unit test coverage exceeds 85% of code paths; regression tests validate calculation stability across script versions
5. User documentation includes methodology explanation, input parameter definitions, output interpretation, and troubleshooting guidance
6. Scripts are deployable via version control with change tracking and rollback capability
